## 说明

该项目基于原生插件 [signature_pad](https://github.com/szimek/signature_pad) 开发, 在它的基础上使用Angular进行封装, 对外提供更加适合Angular风格的API. 该插件只具备很基础的`签名Canvas`的功能, 而`ngx-signature-pad`则提供了一些实际应用场景中的一些例子, 如`旋转签名`等. 

## 注意

该项目尚未全部完成, 尚在研发中.
